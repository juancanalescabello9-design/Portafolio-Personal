Proyectos destacados

Soluciones construidas en entornos industriales reales, no en datasets de Kaggle.

Nota de confidencialidad: Los proyectos descritos aquí fueron desarrollados en empresas. No puedo compartir los archivos originales, pero puedo hablar en detalle sobre metodología, decisiones técnicas y resultados en una entrevista.
01 · Gardilcic / El Teniente
Dashboard de KPIs Gerenciales
Power BI · DAX · Power Query · Excel

Modelo de datos que integra fuentes de distintos sistemas (planillas de terreno, SAP, Excel) y las expone como un único dashboard interactivo para reuniones semanales de superintendencia. Incluye seguimiento de avance físico, estados de pago y proyecciones PF.

Eliminó reportes manuales y consolidó la toma de decisiones de avance en un solo lugar.
02 · Gardilcic / Memoria de Título
Monitor de Calidad en Faena
Power BI · DAX · Power Query

Dashboard para monitoreo en tiempo real de indicadores de desempeño en trabajos de obras civiles mineras. Conectado a fuentes actualizadas periódicamente, con alertas visuales para desviaciones fuera de rango.

Permitió identificar patrones de desvío temprano y actuar antes de que afectaran el avance del programa.
03 · Agrosuper
Automatización de Control de Caducidad
Python · SAP HANA · Automatización

Script Python que se conectaba a la base de datos de producción, identificaba automáticamente productos congelados en riesgo de vencer antes del despacho y generaba reportes de alerta para el equipo de planta.

Reemplazó un proceso manual diario y redujo el riesgo de pérdidas por vencimiento en la cadena de distribución.
04 · Gardilcic / Memoria de Título
Herramienta de Control de Hormigón
HTML · JavaScript · Validación de datos

Aplicación web liviana para registrar y validar consumos de hormigón en terreno, comparando automáticamente contra valores teóricos y marcando desviaciones significativas en tiempo real.

Detectó sobreconsumos de hasta 200%, generando ahorro directo de materiales en faena.
05 · Proyecto Propio · SaaS en Producción
PanApp — Gestión de Pedidos para Panaderías
Next.js 14 · TypeScript · Supabase · Tailwind CSS · Framer Motion · Mercado Pago

SaaS B2B que digitaliza la gestión de pedidos entre panaderías y sus almacenes distribuidores en Chile. Una panadería que despacha a más de 40 almacenes recibía 40 llamadas diarias para coordinar pedidos — PanApp reemplazó ese proceso con un sistema completo: catálogo de productos, carrito inteligente, calendario de entregas por turno AM/PM, y pagos integrados con Mercado Pago.

Incluye dashboard administrativo en tiempo real con KPIs, control de capacidad de producción que bloquea automáticamente pedidos cuando el turno está lleno, planificación de rutas de reparto con Google Maps, y gestión de estados tipo Kanban. Arquitectura multi-tenant, autenticación por RUT, validación de datos (RUT chileno, teléfono, dirección por campos), RLS completo en Supabase, y deploy en Vercel.
